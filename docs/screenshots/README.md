# Screenshots

These are **branded preview frames** (`.svg`) so the README looks finished right
now. Swap in real captures of [slovey.dev](https://slovey.dev) whenever you like —
it's a one-file change per image.

| File | Shows | Suggested capture |
|------|-------|-------------------|
| `landing.svg` | Landing hero | Full-width hero at `slovey.dev` |
| `ask.svg` | Ask Slovey | A real Q&A with a citation chip, in the app |
| `graph.svg` | Decision graph | The org's decision graph view |
| `preflight.svg` | Preflight gate | `slovey preflight` output catching a conflict |

## How to swap in a real screenshot

1. Take the screenshot (a dark-theme capture matches the frames best).
2. Save it over the placeholder, e.g. `docs/screenshots/ask.png`.
3. In the main [`README.md`](../../README.md), change that image's `src`
   from `docs/screenshots/ask.svg` to `docs/screenshots/ask.png`.

That's it — GitHub picks it up on the next push. Recommended size **1200×750**
(16:10) so the grid stays even.
